What is HTML?

A hyperlink (or link) is a clickable element—typically highlighted text or an image—that connects one web page, document, or file to another, facilitating navigation. They are foundational to the internet, allowing users to jump between resources by clicking or tapping. Hyperlinks are created in HTML using the < a href="URL" > tag.

Characteristics of HTML

  1. Uses elements/tags to build web page structure
  2. Emables media files to be embedded in webpages i.e. audio and video clips
  3. Semantically defines web page for text i.e. paragraphs, headings etc.
  4. Allows developers to embed JavaScript code
  5. Elements/tags have attributes
